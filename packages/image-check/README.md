# Install

```bash
yarn add @aelf-design/image-check
```

# Usage

1. add `image-check` command to your `scripts` in `package.json`

```js
...
"scripts": {
  "check": "image-check"
}
...
```

2. run `yarn run check`

```bash
yarn run check
```
